NeoLoad Web can be deployed in many environments, primarily because it is container based.

- [NeoLoad Web in AWS](/tutorials/deploying-neoload-web-in-aws)
- [NeoLoad Web OpenShift Templates](https://github.com/Neotys-Labs/neoload-web-openshift-template)
- [NeoLoad Web Helm Charts for Kubernetes Clusters](https://github.com/Neotys-Labs/helm-neoload-web)
- NeoLoad Web on Windows (not supported)
- NeoLoad Web on Linux (not for high availability)
  -> install Docker and use [Docker Compose files](https://www.neotys.com/documents/doc/nlweb/latest/en/html/#25962.htm)

Don't know how to do any of this or need help? Contact your Neotys CSM representative
 and schedule a session to discuss what you're looking to do.
