You can use NeoLoad to test SAP GUI applications.

# Recording SAP GUI Workflows

As with browser workflows, you can manually record SAP GUI workflows in NeoLoad.

[Please see the official product documentation for more detail.](https://www.neotys.com/documents/doc/neoload/latest/38303.htm)

# Tricentis TOSCA Integration

With the NeoLoad integration to Tricentis TOSCA, your SAP GUI test cases can
 be reused to create highly scalable load tests that are fully customizable and
 data driven, validated and ready in minutes.
